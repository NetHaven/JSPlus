JS+ is a library to add more classical object-oriented features to JavaScript. It supports the creation of class objects, fields, methods, public vs. private scope, constructors, a single-inheritance model, & static elements.

What's New in Version 1.1
This update includes a new package (graphics) and several new classes:

JS.graphics.Dimension - The Dimension class encapsulates the width and height of an object.
JS.graphics.Point - The Point class encapsulates the x and y coordinate of a single object.
JS.lang.Exception - The Exception class encapsulates conditions that an application might want to catch.
JS.lang.System - The System class contains several useful methods that obtain system specific information. It cannot be instantiated.
